# Coffee Machine

A command-line simulation of a coffee machine that accepts drink orders, tracks inventory, processes coins, and returns change.

## What It Demonstrates

- Business logic for a small stateful application
- Inventory and resource checks
- Currency and change handling
- Use of shared configuration data from a helper module

## Run

```bash
python3 projects/utilities/coffee-machine/main.py
```

## Also Included

- `oop_version/` contains an alternate object-oriented version of the same exercise.
- Run it with `python3 projects/utilities/coffee-machine/oop_version/main.py`.
