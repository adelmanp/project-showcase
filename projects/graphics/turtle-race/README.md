# Turtle Race

An interactive `turtle` graphics game where the user places a bet on a turtle color and watches a randomized race play out.

## What It Demonstrates

- Event-driven user input
- Basic graphics with the standard library
- Randomized simulation logic
- Managing multiple objects on screen

## Run

```bash
python3 projects/graphics/turtle-race/turtle_race.py
```

## Also Included

- `etchASketch.py` is a small keyboard-controlled drawing sketch kept with this graphics exercise.
- Run it with `python3 projects/graphics/turtle-race/etchASketch.py`.
